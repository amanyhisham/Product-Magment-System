
# 🗂️ CRUDS — Product Management System

A dynamic **Product Management System** that supports full CRUD operations, built with **HTML**, **CSS**, and **JavaScript**.

---

## 🚀 Live Demo

🔗 [View Live](https://amanyhisham.github.io/Product-Magment-System/)

---

## ✨ Features

- ➕ Create new products with: Name, Price, Taxes, Ads, Discount, Count, and Category
- 🔢 Auto-calculated Total price
- ✏️ Update existing products
- 🗑️ Delete products
- 🔍 Search by Title or by Category
- 📊 Sortable table columns (Title, Price, Taxes, Ads, Discount, Total, Category)
- 🌙 Dark mode UI
- 💾 Data saved in localStorage

---

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript
